```python
from PIL import Image, ImageChops, ImageEnhance
import numpy as np

def compute_ssim(baseline_path: str, current_path: str) -> dict:
    """
    Compare two screenshots using SSIM.
    Returns: { ssim_score, visual_diff_pct, diff_image_path, visual_regression }
    """
```

Logic:
- Load both images, convert to grayscale numpy arrays
- Compute SSIM score (range 0-1, where 1 = identical)
- Compute pixel-level diff using PIL ImageChops.difference
- visual_diff_pct = percentage of pixels with diff > threshold (threshold = 10/255)
- visual_regression = True if ssim_score < 0.98
- Save diff overlay image to `/tmp/ssim-diff-[event_id].png`
- Return all values as a dict

Tune the threshold to achieve < 5% false positive rate.
Write tests using 30 clean-patch screenshot pairs (generate programmatically — 
create slight JPEG compression artifacts to simulate real-world variance).

Tier logic:
